ちなみに書いたコードはこちらになります

```
quickSort :: (Ord a) => [a] -> [a]
quickSort [] = []
quickSort (x:xs) = quickSort [y | y <- xs, y < x] ++ [x] ++ quickSort [y | y <- xs, y > x]
```

- ってドヤ顔しようとしたら、いろんな文字入れると見事に削られてしまった
